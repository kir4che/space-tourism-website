# Space Tourism Website

## 項目展示
[Demo](https://space-tourism-website-33uszmh8e-kir4che.vercel.app/home)（2022.08.06 完成）

## 功能說明
多頁式航空旅遊網站

點擊按鈕可跳轉至相應頁面，且版面也適用於平板與手機。

## 目的
練習 React Router 跳轉

並加強練習 Button、Link 如何搭配 useState 作出點擊後效果
